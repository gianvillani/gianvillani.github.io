Project 1: Java GUI

This Java GUI was a class project that required us to make a "Amazon" like store that interacted with a CSV file and added items to a cart. In this project, the user was allowed to add and remove items from their cart and completely reset their whole order. This project taught me a lot on how powerful java can be sometimes. It was my first time using Java for a GUI but was a fun experience. Here is a list of what I learned while making this, and below will be a button to see the program in action.

  • Java GUI programming (Swing/AWT)
  • Event-driven programming
  • File I/O & CSV parsing
  • Data validation & error handling
